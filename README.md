# SENSI : Efficiency Redefined 🔴

**SENSI** adalah sistem presensi open-source yang dirancang untuk memangkas birokrasi manual menjadi lebih efisien. Proyek ini dibangun sebagai official library sekaligus manual book bagi pengguna yang ingin mengimplementasikan sistem presensi dengan standar visual high-end.

## 🚀 Fitur Utama
* **Minimalist Interface:** Antarmuka bersih dengan tema Cyber-Stealth.
* **Open Source:** Bebas untuk dimodifikasi dan dikembangkan sesuai studi kasus.
* **Figma Integrated:** Langsung terhubung ke prototype sistem utama.

## 🛠️ Tech Stack
* [React.js](https://reactjs.org/) - Frontend Framework
* [CSS3](https://developer.mozilla.org/en-US/docs/Web/CSS) - Custom Modern Styling
* [Figma](https://www.figma.com/) - System Prototyping

## ⚙️ Cara Instalasi (Local Development)

1. **Clone Repository**
   ```bash
   git clone [https://github.com/ryann-bot/sensii.git](https://github.com/ryann-bot/sensii.git)
   cd sensii

2. **Install Dependensi**.
   * Pastikan sudah install Node.js
   ```Bash
   npm install

3. **Running di Lokal**
    ```Bash
   npm start
